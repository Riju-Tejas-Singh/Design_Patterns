```text
                    +------------------------------+
                    |          Singleton           |
                    +------------------------------+
                    | - instance : Singleton       |

                    +------------------------------+
```

#### Factory Method

```text
                         Product
                      <<interface>>
                    +---------------+
                    | + operation() |
                    +---------------+
                           ▲
              ┌────────────┴────────────┐
              │                         │
       +---------------+         +---------------+
       | ProductA      |         | ProductB      |
       +---------------+         +---------------+


                         Creator
                      <<abstract>>
             +---------------------------+
             | + factoryMethod()         |
             +---------------------------+
                           ▲
              ┌────────────┴────────────┐
              │                         │
      +------------------+      +------------------+
      | CreatorA         |      | CreatorB         |
      +------------------+      +------------------+
      | + factoryMethod()|      | + factoryMethod()|
      +------------------+      +------------------+
              │                         │
              │ creates                 │ creates
              ▼                         ▼
          ProductA                  ProductB
```

#### Builder

```text
                        Product
                  +----------------------+
                  | - field1             |
                  | - field2             |
                  | - field3             |
                  +----------------------+
                  | - Product(Builder)   |
                  +----------------------+
                           ▲
                           │ builds
                           │
                  +----------------------+
                  | Builder              |
                  | <<static>>           |
                  +----------------------+
                  | + field1(...)        |
                  | + field2(...)        |
                  | + field3(...)        |
                  | + build()            |
                  +----------------------+
```
